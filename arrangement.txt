산점도 - x, y축으로 이뤄진 좌표계에 두 변수의 관계를 표현하는 방법
맷플롯립(matplotlib) - 파이썬에서 과학계산용 그래프를 그리는 대표적인 패키지
scatter() - 산점도를 그리는 함수, matplotlib패키지를 임포트해야 사용 가능
선형적 - 산점도 그래프가 일직선에 가까운 형태로 나타날 경우
from ~ import 구문 - 패키지나 모듈 전체를 임포트하지 않고 특정 클래스만 임포트하려면 다음 구문을 사용
KNeighborsClassifier - 사이킷런 패키지의 k-최근점 이웃 알고리즘을 구현한 클래스
n_neighbors 매개변수로 이웃의 개수를 지정(기본값은 5)

사이킷런의 fit() - 주어진 데이터로 알고리즘을 훈련
k-최근점 이웃 알고리즘 - 어떤 데이터에 대한 답을 구할 때 주위의 다른 데이터를 보고 다수를 차지하는 것을 정답으로 사용
predict() - 새로운 데이터의 정답을 예측
score() = 훈련된 사이킷런 모델의 성능을 측정

2-1.훈련 세트와 테스트 세트
샘플링 편향 - 훈련 세트와 테스트 세트에 샘플이 골고루 섞여 있지 않으면 샘플링이 한쪽으로 치우쳤다는 의미
넘파이 - 배열 라이브러리
seed() - 넘파이에서 난수를 생성하기 위한 정수 초깃값을 지정
arrange() - 일정한 간격의 정수 또는 실수 배열을 만듦
shuffle() - 주어진 배열을 랜덤하게 섞음

2-2.데이터 전처리
a[:,0] - 모든 행에 대해서 첫번째 열의 정보를 가져다 달라, 가로
a[:,1] - 모든 행에 대해서 두번째 열의 정보를 가져다 달라, 세로
데이터 전처리 - 모신러닝 모델에 훈련 데이터를 주입하기 전에 가공하는 단계
표준점수 - 훈련 세트의 스케일을 바꾸는 대표적인 방법 중 하나, 표준점수를 얻으려면 특성의 평균을 빼고 표준편차로 나눔.
브로드 캐스팅 - 크기가 다른 넘파이 배열에서 자동으로 사칙 연산을 모든 행이나 열로 확장하여 수행하는 기능.
train_test_split() - 훈련 데이터를 훈련 세트와 테스트 세트로 나누는 함수
knerighbors() - k-최근점 이웃 객체의 메서드, 입력한 데이터에 가장 가까운 이웃을 찾아 거리와 이웃 샘플의 인덱스를 반환

3-1.k-1최근접 이웃 회귀
회귀 - 임의의 수치를 예측하는 문제, 타깃값도 임의의 수치가 됨.
결정계수 - 대표적인 회귀 문제의 성능 측정 도구, 1에 가까울수록 좋고 0에 가까우면 성능이 나쁜 모델임.
과대적합 - 모델의 훈련 세트 성능이 테스트 세트 성능보다 훨씬 높을 때 일어남.
과소적합 - 모델이 훈련 세트에 너무 집착해서 데이터에 내재된 거시적인 패턴을 감지하지 못함.
KNeighborsRegressor - k-최근접 이웃 회귀 모델을 만드는 사이킷런 클래스., n_neighbors매개변수로 이웃의 개수를 지정
mean_absolute_error() - 회귀 모델의 평균 절댓값 오차를 계산, 첫 번째 매개변수는 타깃, 두 번째 매개변수는 예측값을 전달
numpy - reshape() - 배열의 크기를 바꾸는 메소드

3-2.선형 회귀
선형 회귀 - 특성과 타깃 사이의 관계를 가장 잘 나타내는 선형 방정식
모델 파라미터 - 선형 회귀가 찾은 가중치처럼 머신러닝 모델이 특성에서 학습한 파라미터
다항 회귀 - 다항식을 사용하여 특성과 타깃 사이의 관계를 나타냄.
LienarRegression - 사이킷런의 선형 회귀 클래스
